1. How many multiplications and additions do you need to perform a matrix multiplication between a (n, k) and (k, m) matrix? Explain.

For multiplication of any two matrices- A(n,k) , B(k,m), we should take the inner product of every row of matrix A with every column of matrix B.

So, for example we take C (n,m) = A(n,k)*B(k,m)

From the above operation, The number of multiplications required for getting one element of C = k

The number of additions required for getting one element of C = k -1

For the entire Matrix Multiplication, we need Total Multiplication operations = Total Number of elements in C X Number of multiplications required for computing a single element in C = nxmxk.

Total Addition operations = Total Number of elements in C X Number of addition operation required for computing a single element in C = nxmx(k-1).
